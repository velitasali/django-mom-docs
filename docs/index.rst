Model Object Mapper for Django
==============================

Welcome to the Model Object Mapper Documentation page!

Model Object Mapper, or *MOM* for short, is a Django Management Utility for
statically creating and updating database entries. It supports foreign fields 
and can work with Django apps without requiring any modification.
